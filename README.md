<h1 align=""> Descrição do Projeto </h1>
Este projeto foi desenvolvido como parte de um desafio proposto pelo curso da Alura em parceria com o Oracle Next Education. O objetivo foi aplicar os conhecimentos adquiridos no curso de Lógica com JavaScript para completar o código de um site responsável por realizar o sorteio de um Amigo Secreto

<h1 align=""> Funcionalidades </h1>
A aplicação permite que os usuários insiram os nomes que irão participar e realizem o sorteio de forma automática e aleatória.
